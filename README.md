### Custom Keybindings
Custom keybinds for all the plugins that I use.

##### System Remaps

| Name                  | Keybind |
| --------------------- | ------- |
| New Buffer            | Ctrl n  |
| Delete Current Buffer | Ctrl b  |
| Next Buffer           | Ctrl m  |

##### Code Wrapping

| Name           | Keybind                                  |
| -------------- | ---------------------------------------- |
| replace quotes | cs[surrounding character][new character] |

##### Commenting

| Name      | Keybind |
| --------- | ------- |
| Comment   | gcgc    |
| UnComment | gcgc    |
